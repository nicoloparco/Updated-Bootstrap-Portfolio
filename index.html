    Friend Finder

    An app in which users fill out a questionnaire to determine with other users they are most similar to. Each users' data is saved in JSON format so that whenever a new user completes the survey they will be shown their closest match.

    A MySQL CLI app that simulates a user shopping experience through interaction with a data table of products. Whenever the user selects a product to "buy" the product table is updated.
